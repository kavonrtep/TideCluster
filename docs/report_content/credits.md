## Credits and citations

TideCluster uses [TideHunter](https://github.com/Xinglab/TideHunter)
for tandem repeat detection and TAREAN for consensus sequence
reconstruction.

If you use TideCluster please cite:

- [kavonrtep/TideCluster](https://github.com/kavonrtep/TideCluster) —
  DOI [10.5281/zenodo.601111441](https://zenodo.org/badge/latestdoi/601111441).
- Novák P. *et al.* TAREAN: a computational tool for identification
  and characterization of satellite DNA from unassembled short reads.
  *Nucleic Acids Research* 45 (2017).
  [doi:10.1093/nar/gkx257](https://doi.org/10.1093/nar/gkx257).
- Gao Y. *et al.* TideHunter: efficient and sensitive tandem repeat
  detection from noisy long-reads using seed-and-chain.
  *Bioinformatics* 35 (2019).
  [doi:10.1093/bioinformatics/btz376](https://doi.org/10.1093/bioinformatics/btz376).
